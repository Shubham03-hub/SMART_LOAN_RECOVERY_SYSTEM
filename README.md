# Smart Loan Recovery System

## Overview
This project predicts loan default risk using Machine Learning and recommends recovery strategies for financial institutions.

## Objective
Identify high-risk borrowers and suggest recovery actions to minimize financial losses.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- SQL
- Flask
- Streamlit

## Features
- Loan default prediction
- Risk scoring system
- Recovery strategy recommendation
- SQL loan data analysis
- Interactive Streamlit dashboard
- Flask API for predictions

## Workflow
Data Collection → Data Cleaning → Feature Engineering → Model Training → Risk Scoring → Recovery Strategy → Dashboard

## Project Structure

data/ → dataset  
src/ → machine learning scripts  
models/ → trained model  
api/ → Flask prediction API  
dashboard/ → Streamlit dashboard  

## Result
The system predicts borrower default risk and suggests appropriate recovery strategies.
